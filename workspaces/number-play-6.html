What is Number Play?

Number Play is about noticing patterns and making smart decisions with numbers. Instead of only calculating, you learn to compare, estimate, arrange, decode, and reason.

Place Value
Each digit has value based on its position.
Number Arrangement
Digits can be rearranged to form greatest or smallest numbers.
Estimation
Find quick approximate answers and compare closeness.
Local Logic
Use immediate neighbours to solve puzzles.

Taller Neighbours Discovery

A child says how many immediate neighbours are taller. End children can never say 2 because they have only one neighbour.

Important Rules

Greatest Number Rule
Arrange digits in descending order.
Smallest Number Rule
Arrange digits in ascending order. Avoid zero at the first place if present.
Closeness Rule
Compare absolute differences from the target.
Missing Addend Rule
Missing number = Target − known sum.
