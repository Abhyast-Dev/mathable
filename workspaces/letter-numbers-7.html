Class VII: Expressions Using Letter-Numbers

Use letters as numbers, build expressions from patterns, evaluate expressions, and describe positions in grids.

Fact Sheet: What are Letter-Numbers?

Letters can stand for numbers. This helps us write rules that work for many cases instead of only one case.

Variable: A letter like r, c, or n that represents a number.
Expression: A mathematical phrase such as 4(r − 1) + c.
Grid Rule: In a 4-column grid, the number in row r and column c is 4(r − 1) + c.
